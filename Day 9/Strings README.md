Java String Class:
String is a class in Java (java.lang.String)
It is used to store text (sequence of characters)
Strings in Java are immutable (cannot be changed)

Ways to create a String:
String s1 = "Java";              // String Literal
String s2 = new String("Java");  // Using new keyword

Key Points:
equals() compares content
== compares reference
String literals are stored in String Constant Pool (SCP)

Commonly Used Methods:
length()
toUpperCase()
toLowerCase()
charAt()
equals()

Taking String Input from User:
Scanner sc = new Scanner(System.in);
String name = sc.nextLine();
